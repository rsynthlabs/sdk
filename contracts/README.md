# contracts

solidity contracts for the rsynth execution log. anchors signed payload hashes on base.

## build

```bash
forge build
```

## test

```bash
forge test
```

## deploy

```bash
forge script script/Deploy.s.sol --rpc-url <rpc> --broadcast
```

see [`../SCHEMA.md`](../SCHEMA.md) for the on-chain anchor contract spec.
